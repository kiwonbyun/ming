# @b-origin/ming-toast

A React toast component library.

## Installation

```bash
npm install @b-origin/ming-toast
# or
yarn add @b-origin/ming-toast
# or
pnpm add @b-origin/ming-toast
```

## Usage

```jsx
import { toast, Toaster } from "@b-origin/ming-toast";

function App() {
  return <Toaster />;
}
```
