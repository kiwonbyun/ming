# @wemeet-overlay/toast

A React toast component library.

## Installation

```bash
npm install @wemeet-overlay/toast
# or
yarn add @wemeet-overlay/toast
# or
pnpm add @wemeet-overlay/toast
```

## Usage

```jsx
import { toast, Toaster } from "@wemeet-overlay/toast";

function App() {
  return <Toaster />;
}
```
